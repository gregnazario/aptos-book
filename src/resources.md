# Learning Resources

## Official Documentation

- [Aptos Developer Docs](https://aptos.dev) - The official developer documentation
- [Aptos Learn](https://learn.aptoslabs.com) - Interactive tutorials and courses
- [Move Language Reference](https://aptos.dev/en/build/smart-contracts/book) - The Move language specification

## Tools

- [Aptos CLI](https://aptos.dev/en/build/cli) - Command-line tool for building, testing, and deploying
- [Aptos Explorer](https://explorer.aptoslabs.com) - Block explorer for viewing transactions and accounts
- [Aptos TypeScript SDK](https://github.com/aptos-labs/aptos-ts-sdk) - TypeScript/JavaScript SDK for building dApps

- [Aptos Discord](https://discord.gg/aptosnetwork) - Community chat and developer support
- [Aptos GitHub](https://github.com/aptos-labs) - Source code and open-source projects
- [Aptos Forum](https://forum.aptoslabs.com) - Governance and technical discussions
